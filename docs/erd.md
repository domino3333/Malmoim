# 말모임 erd

## 1. 시나리오




호스트는 방을 만들고, 방 코드를 부여받는다
호스트는 방 코드를 참여자들에게 나눠준다 < 이건 뭘로 나눠줌?(링크와 함께 코드가 전달되도록 하면 좋을 듯)
참여자들이 방에 입장하는 순간 모달 창으로 카카오톡 오픈채팅방처럼 익명의 닉네임을 설정할 수 있다
근데 이 순간 각 사용자는 db에 어떻게 저장될 것인지, 아니면 그냥 redis로만 저장할 것인지 생각을 해야 함

참여자들은 질문하기 버튼을 눌러나오는 모달창에 n자 이내의 문장으로 질문을 할 수 있다.
질문을 등록하는 순간 호스트와 다른 참여자들의 화면에도 즉시 질문글이 나타나며
질문글 옆에는 추천하기(엄지 이모지)가 생기며 추천수에 따라 질문글들의 순위가 재정렬된다.

관리자는 추천수대로 정렬된 질문글들을 볼 수 있으며 답변을 완료한 질문에 대해서는 완료/숨기기/답변거부 버튼을 눌러
해당 질문을 완료처리할 수 있다.
이 또한 모든 사용자에게 즉시 반영된다.
